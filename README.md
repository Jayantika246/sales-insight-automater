# Sales Insight Automator

A frontend prototype for uploading sales data files.

## Setup

```bash
npm install
npm start
```

The app will open at http://localhost:3000

## Features

- File upload (.csv and .xlsx)
- Email input
- Simulated API call with loading state
- Error handling for unavailable backend

## Note

This is a frontend-only prototype. The backend API endpoint does not exist, so all submissions will show an error message after a brief loading period.
